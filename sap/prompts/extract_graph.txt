-Goal-
Given SAP “What’s New” release notes and a list of entity types, extract entities and relationships.

-Steps-
Entity extraction
• Identify relevant entities such as:
  – SAP products, services, or components (e.g., Authorization and Trust Management Service, Neo)
  – Technical objects (e.g., Root Certificate, instance-secret credential type)
  – Organizations (e.g., DigiCert)
  – Geographies, dates, or other provided entity types [{entity_types}]
• For each entity, output:
  – entity_name: Uppercased
  – entity_type: One of [{entity_types}]
  – entity_description: ≤25 words, SAP context and lifecycle impact
• Format:
("entity"{tuple_delimiter}<entity_name>{tuple_delimiter}<entity_type>{tuple_delimiter}<entity_description>)

Relationship extraction
• From identified entities, extract pairs where the text shows a clear dependency, lifecycle change, or impact.
• For each relationship, output:
  – source_entity
  – target_entity
  – relationship_description: ≤20 words, SAP impact
  – relationship_strength: Integer 1–10 (1 = weak, 10 = strong)
• Format:
("relationship"{tuple_delimiter}<source_entity>{tuple_delimiter}<target_entity>{tuple_delimiter}<relationship_description>{tuple_delimiter}<relationship_strength>)

-Output rules-
• Return all entities and relationships in a single list
• Separate items with {record_delimiter}
• End with {completion_delimiter}

-Constraints-
• Do not fabricate SAP changes
• Only tie entities and relationships to the actual lifecycle, component, or action in text

-Minimal Examples-
Example 1
Entity_types: ORGANIZATION, TECHNOLOGY
Text:
SAP BTP Neo will switch to DigiCert Global Root G3 for all platform domains by April 2026.
Output:
("entity"{tuple_delimiter}SAP BTP NEO{tuple_delimiter}TECHNOLOGY{tuple_delimiter}SAP environment subject to root certificate replacement)
{record_delimiter}
("entity"{tuple_delimiter}DIGICERT GLOBAL ROOT G3{tuple_delimiter}ORGANIZATION{tuple_delimiter}Root certificate authority replacing DigiCert Global Root G2)
{record_delimiter}
("relationship"{tuple_delimiter}SAP BTP NEO{tuple_delimiter}DIGICERT GLOBAL ROOT G3{tuple_delimiter}Neo environment certificates will be signed by DigiCert Global Root G3{tuple_delimiter}9)
{completion_delimiter}
