Component: Extensions
Environment: Cloud Foundry,Kyma
Title: Use a New Formation of Type Integration with Joule
Description: You can use Joule in SAP systems. To do that, you have to integrate these systems with Joule by including them in a formation of type Integration with Joule. The integration is between the Joule system and every SAP system in the formation, but not between each and every SAP system.  See Enabling Joule.

--- Record 1610 ---
Component: SAP Build Work Zone, advanced edition
Environment: Cloud Foundry
Title: Multiple Subscriptions to SAP Build Work Zone Editions
Description: It's no longer possible to subscribe to multiple SAP Build Work Zone editions in parallel in a given SAP BTP subaccount. Trying to subscribe to multiple editions, for example both to standard and advanced editions in the same subaccount, will result in an error.   Note that existing subscriptions are not impacted by this restriction and continue to work as before.
Action: Info only
Lifecycle: General Availability
Type: Changed
Line of Business: Technology
