--- Record 3803 ---
Component: Mobile Services
Environment: Cloud Foundry
Title: Offline Defining Requests - Token Lifetime
Description: (Version 2311) The default Token Lifetime expiration value has been changed from 72000 minutes (about 50 days) to 21600 minutes (about 15 days). This value controls how long the oldest change log data are retained. The current value is not changed if you explicitly set it, but the new default appears when you add a new defining request. This change improves database usage and performance of the mobile service.  See Editing the Application Configuration File
