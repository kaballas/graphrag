--- Record 719 ---
Component: API Management
Environment: Cloud Foundry
Title: Support for API Artifacts Added to Access Policies
Description: Support for API artifacts has now been added to access policies. With access policies, unauthorized users can view protected resources—such as integration flows—but cannot perform actions like editing, copying, downloading, deleting, simulating the flow, or monitoring its message content. For more information, see Access Policies for API Artifact and Defining Access Policies.
