--- Record 2235 ---
Component: SAP Build Process Automation
Environment: Cloud Foundry
Title: Additional Property Required for Using SAP Task Center
Description: If you are using SAP Task Center together with SAP Build Process Automation, you must set an additional property in your Identity_Authentication_Connectivity_IDS destination as follows:    If SAP Build Process Automation and SAP Task Center are in the same subaccount, set the wfs.tc.local property to true.      If SAP Build Process Automation and SAP Task Center are in different subaccounts, set the wfs.tc.local property to false.      For more information, see SAP Note 3522594 .  If you don’t set this property, the end users might be able to access their SAP Build Process Automation tasks, but some actions might not be available in the Web app of SAP Task Center.
Action: Required
Lifecycle: General Availability
Type: Changed
Line of Business: Technology
