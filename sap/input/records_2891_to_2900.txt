--- Record 2896 ---
Component: SAP Omnichannel Promotion Pricing
Environment: Cloud Foundry
Title: New Attributes for Identifying Last Calculation-Relevant Changes in Regular Prices and Promotions
Description: You can now use the new attribute lastCalcRelevantChange that is available for both regular prices and promotions. With this attribute that represents a timestamp, you can:     search and select all regular prices with calculation-relevant changes after a specific point in time,       search and select promotions with calculation-relevant changes after a specific point in time.      In the case of regular prices, the attribute is updated when a calculation-relevant change is made using the Data Upload service.  In the case of promotions, the attribute is updated when a calculation-relevant change is made using the Data Upload service or on the Maintain Promotions UI.  For more information about the use cases and examples, see Data Access.
