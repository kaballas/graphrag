--- Record 3775 ---
Component: SAP Digital Payments Add-On
Environment: Cloud Foundry
Title: Additional inbound IP addresses may be used
Description: Additional inbound IP addresses (for communication from any IP address to SAP digital payments add-on) may be used in the EU10 region as of early 2024.   Action: If you manage your firewall using an allowlist, you should ensure that these new addresses are included in your allowlist. For the full current list, see Regions and API Endpoints Available for the Cloud Foundry Environment. In the table Regions for Enterprise Accounts, filter in the column Technical Key for cf-eu10. You will find the relevant IP addresses in the column LB IPs. Note that the SAP digital payments add-on does not use the extensions cf-eu10-002, cf-eu10-003, or cf-eu10-004 so you can omit the IP addresses mentioned there.
