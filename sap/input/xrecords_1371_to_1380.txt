--- Record 1375 ---
Component: Kyma Runtime
Environment: Kyma
Title: Istio module: Update to version 1.14.1
Description: Restarting sidecars is now divided into two phases. In the first phase, only Kyma workloads are restarted. If this phase fails, the Istio Custom Resource (CR) is set to the Error state. In the second phase, customer workloads are restarted in chunks. If any iteration of this phase fails, the Istio CR is set to the Warning state.  We have increased global metrics rotation interval to 6 hours.
