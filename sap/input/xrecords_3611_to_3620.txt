Component: SAP BTP Command Line Interface
Environment: ABAP,Cloud Foundry,Kyma,Other
Title: Management of API Credentials for Calling REST APIs of SAP Authorization and Trust Management Service
Description: We’ve added commands to manage API credentials, which enable developers to access the REST APIs of the SAP Authorization and Trust Management service for all account types. Using btp create security/api-credential, you can create API credentials. It is also possible to list the available API credentials, display their details, and delete them. See Managing API Credentials for Calling REST APIs of SAP Authorization and Trust Management Service.  Action: To use these new features, you need to use version 2.38 of the btp CLI client (see SAP Development Tools).
