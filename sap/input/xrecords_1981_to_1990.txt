--- Record 1985 ---
Component: Cloud Identity Services
Environment: Cloud Foundry,Kyma,Neo
Title: Changed behavior of skip create for SAP Application Server ABAP
Description: The behavior of SAP Application Server ABAP target transformation that contains skip operations for create has been changed.  Previously, Identity Provisioning used to update the resolved users and to provision new users into the target system. After the change, Identity Provisioning updates only users resolved by userName in the target system, while the rest of the users are marked as skipped.  For more information, see Transformation Expressions → skipOperations.
Action: Info only
Lifecycle: General Availability
Type: Changed
Line of Business: Technology
