Title: Pre-Production – Stateless Pricing – Dynamic Scale Base Types in Price Agreements
Description: Price agreements in stateless pricing now support dynamic scale base types as part of the pricing input. Consuming applications can specify the scale base type at runtime, for example, quantity, value, weight, volume. --- Record 28 ---
Component: Pricing Service
Environment: Cloud Foundry
Title: Pre-Production– On-Request Tracing
Description: Additionally to the existing trace settings ON and OFF, a new setting ON_REQUEST was introduced to collect traces only for a subset of requests. To determine for which requests the traces will be collected, the header parameter x-cps-trace needs to be set. The relevant trace type must be set as a parameter value: CONDITION_DETERMINATION, CONDITION_CALCULATION, EXTENSION, or TAXES_US. --- Record 30 ---
Component: Variant Configuration
Environment: Cloud Foundry
Title: Pre-Production - Configurations API – Dependency Profiling Trace
Description: With the dependency profiling trace, it is possible to measure the runtimes of dependencies to analyze the model performance. For on-request tracing, the header parameter x-cps-trace: DEPENDENCY_PROFILING needs to be used. Dependency execution runtimes and counts are recorded and displayed in the administration UI for the configuration service in the section Configuration  Trace   Dependency Profiling Trace.  Dependency profiling is unavailable for products processed through AVC Forwarding. Only dependencies executed within the native engine of the Variant Configuration service will be profiled.  See: Development Guide for SAP Variant Configuration and Pricing, and Administration Guide for SAP Variant Configuration and Pricing.
