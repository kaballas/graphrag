Component: SAP AI Launchpad
Environment: Cloud Foundry
Title: Superdomain URL for SAP AI Launchpad
Description: SAP AI Launchpad has moved to new superdomain URL.SAP is migrating its products to a common cloud.sap superdomain to address browser isolation and privacy issues caused by current domain-specific URLs. This change aims to improve integration scenarios using iFrames or cross-origin requests, which are affected by modern browser security features For existing customers with domain specific URLs, both the original URL and the new superDomain URL will be supported for all clusters.
