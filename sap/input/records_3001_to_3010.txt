Title: Factory Calendar Pull Integration
Description: You can now integrate the S/4HANA Cloud data from your factory calendar into the SAP BTP, ABAP environment to keep them in sync. A pull integration is available that pulls the data from the SAP S/4HANA system and inserts it in the SAP BTP, ABAP environment. Doing this, you can avoid double maintenance, and factory calendar calculations can still be done locally for optimal performance.  For more information, see Integrating Factory Calendar.
